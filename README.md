# 메뉴 리스트

메뉴 항목을 수집하고 관리하는 심플한 웹 앱입니다.

## 기능

- 메뉴 항목 추가 / 삭제
- 항목별 수량 조절 (+ / -)
- 항목 이름 인라인 편집 (클릭하여 수정)
- 총 종류 / 총 수량 표시
- 이전 목록 히스토리 보관
- 새로고침해도 데이터 유지 (localStorage)

## 기술 스택

- React 18
- Vite
- Tailwind CSS

## 로컬 실행

```bash
npm install
npm run dev
```

빌드:

```bash
npm run build
```

## 배포

Vercel을 통해 배포됩니다. `main` 브랜치에 푸시하면 자동으로 배포됩니다.
